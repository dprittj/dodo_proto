This folder contains the DODO basic html prototype for which later functionality and advanced frameworks will be built. This shall serve as a DODO version 0.

Features:
Login/logout functionality for users.
GoogleMapsAPI with geolocation.
Saved profile preferences.
Weather/Calendar (date/time) displays.
Location sourcing based off of user selected interests.
Itinerary generation of locations with user time and cost locations.
Saved location history.
